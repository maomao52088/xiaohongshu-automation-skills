# 小红书内容自动化套件 - 使用说明

## 这是什么

4个AI指令包，帮你自动化小红书内容创作流程。

## 谁适合用

- 小红书博主（个人成长/效率工具/AI测评方向）
- 内容运营（需要批量产出）
- 想做副业但没时间写内容的人

## 怎么用

### 方式1：在Hermes Agent中使用

直接说触发词，自动加载：

| 你说... | 加载指令包 |
|---------|-----------|
| "帮我挖选题" / "小红书选题" | `xiaohongshu-trend-mining` |
| "写小红书文案" / "生成文案" | `xiaohongshu-copywriting` |
| "封面图" / "小红书封面" | `xiaohongshu-cover-prompt` |
| "分析数据" / "笔记数据复盘" | `xiaohongshu-data-analysis` |

**示例对话**：
```
你：帮我挖AI工具方向的选题
我：[加载选题挖掘] → 输出10个选题推荐

你：写第一篇文案
我：[加载文案生成] → 输出完整小红书文案

你：生成封面提示词
我：[加载封面提示词] → 输出Midjourney/DALL-E提示词
```

### 方式2：在其他AI工具中使用

**Claude / ChatGPT**：
1. 打开对应的SKILL.md文件
2. 复制全部内容
3. 粘贴给AI，说"按照这个流程执行"

**示例**：
```
用户：[粘贴 xiaohongshu-copywriting 内容]
用户：用这个模板写一篇关于Hermes Agent的文案
```

### 方式3：批量工作流

每周一运行完整流程：

```
周一上午：
1. "帮我挖本周选题" → 得到10个选题
2. 选3个，让AI生成3篇文案
3. 让AI生成3组封面提示词
4. 用Midjourney生成封面图

周一晚上：
5. 发布

周六：
6. "分析最近10条笔记数据" → 得到优化建议
```

## 指令包详细说明

### 1. xiaohongshu-trend-mining（选题挖掘）

**输入**：领域关键词（如"AI工具"、"个人成长"）

**输出**：10个选题 + 爆款分析 + 优先级排序

**流程**：
1. 确定领域关键词
2. 分析爆款共性（标题公式、封面特征）
3. 生成选题清单
4. 验证可行性

---

### 2. xiaohongshu-copywriting（文案生成）

**输入**：选题 + 内容类型（干货/故事/清单）

**输出**：完整小红书文案 + 数据预测

**模板**：
- 干货型：高收藏，适合教程
- 故事型：高互动，适合人设内容
- 清单型：高转发，适合工具推荐

---

### 3. xiaohongshu-cover-prompt（封面提示词）

**输入**：内容类型 + 风格偏好

**输出**：AI绘图提示词（Midjourney/DALL-E可用）

**类型**：
- A：人物+文字（个人成长）
- B：对比图（测评类）
- C：纯文字排版（清单类）
- D：场景+道具（生活方式）

---

### 4. xiaohongshu-data-analysis（数据分析）

**输入**：最近10条笔记数据

**输出**：分析报告 + 优化建议

**指标**：
- 收藏率 > 10% = 干货型爆款
- 评论率 > 5% = 高互动内容
- 分享率 > 3% = 传播潜力

## 常见问题

**Q：指令包要钱吗？**
A：免费，开源，随便用。

**Q：要会编程吗？**
A：不用。会打字、会用AI聊天工具就行。

**Q：支持哪些AI工具？**
A：理论上所有AI都能用（Claude、ChatGPT、Gemini、DeepSeek等）。Hermes Agent用户可以直接触发。

**Q：可以修改指令包吗？**
A：可以。SKILL.md是markdown文件，随便改。建议根据你的领域定制。

**Q：数据来源可靠吗？**
A：爆款规律基于路标笔记实测数据（CTR 22.6%，完播率36%）。
